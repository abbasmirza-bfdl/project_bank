ABOUT US

Banky Bank is an experimental software project for baking operations. This project has been created while we were learning Django. We have used Python, Django, PostgreSQL, Swagger API, JavaScript, HTML and CSS. We have built this by collaborating on GitHub and coded on Visual Studio Code.

Our demo bank is a financial institution that gets customers to register with us. We create an account for them. We can deposit, withdraw, update, delete users and watch transaction history. Our portal is secure and authentication & authorization is required for any operations to be done.
